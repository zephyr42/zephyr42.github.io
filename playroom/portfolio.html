StoryMap JS — The Marble Faun

Spring 2014

For LSC874 Digital Humanities, I created two different ways of viewing Nathaniel Hawthorne's book The Marble Faun. This is the geographical method created using KnightLab's StoryMap JS. The Storymap JS tool includes a construction tool which saves the work on a Google Drive so the project is always accessible and easy to edit.

See the StoryMap (Under Maintenance)

Timeline JS — The Marble Faun

This is the linear method of viewing Hawthorne's story. It was created using Knightlab's Timeline JS tool. The data is stored in a spreadsheet on my GoogleDrive which is parsed through the tool and rendered in a timeline. It's pretty cool.

See the Timeline

Storify

I use Storify to preserve the Twitter feeds of conference I've attended and events I've watched. It initially began as a way to preserve Glen Weldon's #DuringTheGovernmentShutdown2PigInTheCity tweets (the government shutdown as a contained apocalypse) and quickly escalated into a full-on obsession.
